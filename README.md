# astrocircular-skywaver-for-nina
Generates circular defocused star position patterns and N.I.N.A. Advanced Sequencer files for InnovationForesight SkyWave (SKW) telescope collimation. Includes a PixInsight integration script for combining multi-position sub-frames into a single FITS for field-dependent wavefront analysis.
